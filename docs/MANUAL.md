# 操作マニュアル

## 起動方法
```bash
docker compose up --build
```

ブラウザで http://localhost:3000 にアクセス。

## 基本操作

### 指摘を登録する

**部材を指定する場合（Element指摘）**

1. APS Viewerで対象の部材をクリックして選択（青くハイライト）
2. 右パネルの「指摘を登録」フォームに内容を入力
3. 「登録」ボタンをクリック

**空間を指定する場合（Space指摘）**

1. APS Viewerで対象の空間をダブルクリック
2. 右パネルの「指摘を登録」フォームに内容を入力
3. 「登録」ボタンをクリック

> 画面左下に「クリック: 部材選択 / ダブルクリック: 空間指摘」と表示されています。

---

### 指摘の状態を更新する

指摘には3つの状態があり、**一方向のみ**遷移できます。
```
Open（未対応）→ InProgress（対応中）→ Done（完了）
```

| 状態 | バッジ色 | 説明 |
|------|---------|------|
| Open | グレー | 未対応の指摘 |
| InProgress | 黄色 | 対応中 |
| Done | 緑 | 完了（以降変更不可） |

**操作手順**

1. 指摘一覧から対象の指摘をクリック
2. 詳細パネルの「対応開始」または「完了」ボタンをクリック
3. Done になると「完了した指摘のステータスは変更できません」と表示され、ボタンが非表示になります

---

### 写真をアップロードする

1. 指摘詳細パネルを開く
2. 「写真追加」ボタンをクリック
3. ファイル選択ダイアログから画像を選択
4. サムネイルをクリックするとライトボックスで拡大表示

> 是正前・是正後のラベルは登録時に設定できます。

---

### BIMモデル上の位置に移動する

1. 指摘一覧から対象の指摘をクリック
2. APS Viewerのカメラが自動的に指摘位置に移動します

## よくある質問

**Q: APS Viewerにモデルが表示されない**

APS Client ID / Client Secret が正しく設定されているか確認してください。
```bash
# .env を確認
cat .env | grep APS
```

**Q: 写真アップロード後に画像が表示されない**

MinIO が正常に起動しているか確認してください。
```bash
docker compose ps
# storage サービスが Up になっているか確認
```

http://localhost:9001 にアクセスしてMinIOコンソールが開けるか確認してください。
（ユーザー名: `minioadmin` / パスワード: `minioadmin`）

**Q: Done にした指摘を戻せない**

仕様です。完了した指摘の状態は変更できません。
